Research question

When does a collection of experiences deserve to become a belief?

Standard LLM stacks freeze knowledge in weights, lack durable episodic memory, and cannot learn structurally from failure. RAG retrieves documents but has no explicit belief states or feedback loops. ACME operationalizes a full cognitive lifecycle beyond retrieval.

⚠

LLM limitations

No durable memory, no belief promotion, no structured forgetting, unchecked self-confidence, weak learning from outcomes.

◈

ACME response

Graph + vector hybrid memory, CRS belief engine, contrarian verification, compression, forgetting tiers, and autonomous hypothesis generation.

System design

Seven engines, one cognitive loop

Experiences flow through extraction into PostgreSQL (episodic + pgvector) and Neo4j (semantic graph). Queries combine graph neighborhood retrieval with vector search, then optional contrarian challenge. Feedback updates beliefs, logs failures, and triggers consolidation.

Belief lifecycle

Observation → Inference → Hypothesis → Belief → Challenged → Deprecated → Archived. Symmetric demotion on contradictory evidence.

Σ

Cognitive Reliability Score

CRS = 40% prediction success + 20% temporal stability + 20% contradiction resistance + 20% source diversity.

⊘

Multi-tenant isolation

Postgres rows and Neo4j nodes keyed by tenant_id. Per-scenario benchmark sandboxes for reproducible evaluation.

Scenario coverage

Retention, contradiction handling, multi-source conflict, error injection, long-horizon noise, hallucination resistance, feedback/belief adjustment, adversarial noise, tenant isolation, healthcare transfer, multi-session recall, prediction-outcome loop.

Why ACME wins overall

Baselines match ACME on retention-only metrics (~0.96). ACME's advantage comes from feedback correction (1.0) and belief quality (0.7) — dimensions no baseline implements.

Project delivery

Everything built in this release

Core cognitive engine

FastAPI orchestrator with seven specialized engines, event store, hybrid retrieval, contrarian pass, and autonomous learning loop.

  • Belief promotion/demotion with CRS and causal edge types
  • Compression, forgetting tiers (HOT → DELETE), prediction validation
